# IMPULSA – Plataforma Web de Tutorías Académicas

![React](https://img.shields.io/badge/React-17.0.2-blue) ![JavaScript](https://img.shields.io/badge/JavaScript-ES6-yellow) ![Supabase](https://img.shields.io/badge/Supabase-Backend-green)

## Descripción

IMPULSA es una aplicación web desarrollada con **React** orientada a la gestión de tutorías académicas.  
La plataforma permite que universidades e institutos postulen para ofrecer clases de apoyo o cursos y que los estudiantes accedan a reforzamientos o aprendizaje complementario.  

Centraliza la información y facilita la interacción mediante formularios, dashboards y navegación dinámica entre secciones, simulando el flujo real de una plataforma educativa moderna.

---

## Actores del sistema

* **Instituciones educativas:** Postulan y gestionan tutorías.  
* **Estudiantes:** Solicitan apoyo académico en áreas específicas.  
* **Administradores:** Supervisan y validan las solicitudes de tutorías.

---

## Objetivo del proyecto

Desarrollar una interfaz funcional y dinámica con React que permita:  

* Registro e inicio de sesión de usuarios con roles distintos.  
* Validación de formularios y manejo de errores.  
* Navegación fluida entre secciones y dashboards.  
* Interacción con la base de datos mediante Supabase.  

---

## Tecnologías utilizadas

* **Frontend:** React, JavaScript (ES6+), HTML5, CSS3  
* **Backend / Base de datos:** Supabase  
* **Herramientas adicionales:** Node.js, NPM, GitHub  

---

## Funcionalidades principales

* Registro e inicio de sesión de estudiantes e instituciones.  
* Formularios con validación de datos y mensajes de error.  
* Dashboard de seguimiento académico.  
* Solicitud y gestión de tutorías.  
* Visualización de historial de interacciones y estadísticas básicas.  
* Navegación dinámica entre secciones usando React Router.
